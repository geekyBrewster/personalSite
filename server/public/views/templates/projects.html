myBrü - Homebrew Tracking App

Project Description

myBrü is a full-stack application where homebrewers can track the details of the brewing process from recipe preparation through the brewing of the beer and finally to the tasting and evaluation of the final product. The brewer will be prompted to create an account where they’ll store their recipes, brewing notes, & final project notes.

Eventually, myBrü will include additional tracking for fermentation, bottling / kegging, and lagering. There will also be Google Calendar integration with reminders of approaching dates for switching to secondary fermentation, bottling, and drinking the final brew.

Technologies Used

  • MEAN Stack: MongoDB, Express, AngularJS, Node.js
  • Angular Material for styling
  • Angular x-editable for in-line editing
  • Passport.js for user authentication
  • BreweryDB API for beer style data
  • Google Font and Font Awesome for text and icon styling
  • Heroku and mLabs for web deployment
  • git & Github for version control
  • Trello for project management

Goal Tracker CI -
Group project for a local nonprofit organization

Project Description

[Group project] Goal Tracker CI is a full-stack application that will allow the staff at Choice’s three locations to track the personal and professional goals of the participants in their day program services, while also providing data on goal completion over a set period of time. At the basic staff level, users will be able to enter morning & afternoon goal completion, view previous days’ entries, and mark a participant absent. Case managers will also be able to create & edit personalized goals for participants and review summary data for three month & six month periods in order to write reports. Location administrators will be able to manage staff, including add & delete staff accounts; manage participants, including adding & removing participants and assigning case managers; and managing job site information.

Technologies Used

  • SEAN Stack: PostgreSQL, Express, AngularJS, Node.js
  • Angular Material for styling
  • Material Design Data Table for data table styling and pagination
  • Passport.js for user authentication
  • SweetAlert2 for user notifications and modals
  • Font Awesome for icons
  • Heroku for web deployment
  • git & Github for version control
  • Trello for project management
  • Nodemailer for email notifications
  • apiDoc for route documentation
